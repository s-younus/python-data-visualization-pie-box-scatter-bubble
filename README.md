# Canada Immigration Data Visualization

## Project Overview

This project explores immigration trends to Canada using Python data visualization techniques. The analysis focuses on creating clear and meaningful charts to compare immigration patterns by country, continent, and year.

The notebook demonstrates how to prepare data, group records, calculate totals, and build different visualization types including pie charts, box plots, scatter plots, and bubble plots.

## Project Files

- `pie_box_scatter_bubble_plots_clean_github.ipynb` — Jupyter Notebook containing the full analysis and visualizations
- `Canada.csv` — dataset file required to run the notebook locally

## Dataset

The dataset contains immigration records to Canada from multiple countries over several years. The analysis uses yearly immigration values to identify trends, compare regions, and visualize changes over time.

Main data fields include:

- Country
- Continent
- Region
- Development status
- Annual immigration values
- Total immigration count

## Tools and Libraries Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## Visualizations Created

### Pie Charts

Pie charts were used to show the proportion of total immigration to Canada by continent. This helps compare the contribution of different continents to overall immigration.

### Box Plots

Box plots were used to compare immigration distributions across selected countries and time periods. These charts help identify spread, median values, and possible outliers.

### Scatter Plots

Scatter plots were used to show immigration trends over time. A regression line was also added to support trend interpretation.

### Bubble Plots

Bubble plots were used to compare immigration trends between countries while adding size as an additional visual dimension.

## How to Run This Project

1. Download or clone this repository.
2. Make sure the following files are in the same folder:

   ```text
   pie_box_scatter_bubble_plots_clean_github.ipynb
   Canada.csv
   ```

3. Open the notebook in Jupyter Notebook, JupyterLab, VS Code, or Google Colab.
4. Install the required libraries if needed:

   ```bash
   pip install pandas numpy matplotlib
   ```

5. Run the notebook cells from top to bottom.
